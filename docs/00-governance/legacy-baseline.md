# 历史项目遗留基线台账

## 目的

用于登记历史项目暂时无法立即修复的规则豁免项，并设置清退时间，防止豁免永久化。

## 使用规范

- 每条豁免必须有责任人、原因、风险、到期时间。
- 每个迭代必须回顾并压降豁免数量。
- 到期未处理的豁免应升级为阻塞项。
